Обновления в версии 1.5 (11 августа 2022 г.)
Добавлено:
- свойство ActiveView в класс Application, получающее текущий активный интерфейс Renga.IView (видовой интерфейс);
- класс View (представляет отдельный видовой экран/интерфейс), в Renga API есть методы работы только с моделью - IModelView;
- класс ModelView (вид модели и управление видимостью и отоюражением объектов в модели). Реализация соответствующего интерфейса https://help.rengabim.com/api/interface_i_model_view.html;
- класс Other.Camera3D - для работы с камерой (точкой обзора 3д-сцены в Модели). Реализация интерфейса Renga.ICamera3D;
- нод (экспериментальный) DynObjects.Generals.Objects3dTypes со списком Guid-идентификаторов объектов модели, для которых есть 3д-геометрическое представление.

Обновления в версии 1.4.1 (11 августа 2022 г.)
Добавлено:
- в класс PropertyManager добавлены ноды AssignPropertyToTypes, UnassignPropertyFromTypes позволяющие работать со списками категорий объектов;
- в класс Property добавлен метод GetPropertyTypeByString, позволяющий получить тип свойства (Renga.PropertyType) по его строковому наименованию;


Обновления в версии 1.4 (7 августа 2022 г.)
Изменено:
- все поля классов, подразумевающие свойства класса заменены на поля вместо методов (т.е. в Dynamo они будут отображаться как синенькие ноды со значком "вопроса" как свойства, а ноды, которые что-то вычисляют/считают -- остались с "молниями" - действия;
Добавлено:
- класс Room в группе DynObjects (интерфейс Renga.IRoom);
Удалено:
- свойство IModel из Rroject;
- свойства ObjectTypeS, UniqueIds из ModelObject;
- метод GetGuidsS из Other.GuidCollection;




Обновления в версии 1.3 (1 августа 2022 г.)
Изменено:
- описания к большинству нодов;
Добавлено:
- новые ноды в классе Selection: GetSelectedObjectsIdByUser (получает набор численных-int идентификаторов объектов модели, выделенных Пользователем в Renga) и SelectObjectsInModelByIds -- для выделения в модели объектов по списку их численных-int идентификаторов;
- новый нод в класс ExportedObject3D -- GetTrianglesCount для получения количества граней триангуляции в данном объекте модели;
- во все классы добавлен нод CheckIsNotNull, проверяющий на null полученный интерфейс (актуально для нодов List.FilterByBoolMask);
Удалено:
- некоторые ноды, возвращающие строки (при существующих методах возвращающих Guid);



Обновления в версии 1.2 (20 июля 2022 г.)
Добавлено:
- нод GetSelectedObjectsIdByUser для получение int-идентификаторов объектов модели, выбранных Пользователем в параллельном процессе Renga;
- классы Placement2D и Placement3D, описывающие локальную систему координат Renga в двухмерном и трехмерном пространстве;
- класс Placement3Dollection для получения спсика интерфейсов Placement3D;
- класс ColumnParams для получения расширенных параметров о Колонне (типе объекта в Renga);
- новая категория нодов DynDocument.StylesManager для работы с менеджерами стилей объектов Renga куда добавлены классы: BeamStyleManager, ColumnStyleManager, EquipmentStyleManager, PlumbingFixtureStyleManager, ReinforcementUnitStyleManager, SystemStyleManager; классы обеспечивают доступ к стилям;
- новая категория нодов DynStyles с перечнем классов: BeamStyle, ColumnStyle, EquipmentStyle, PlumbingFixtureStyle, RebarStyle, ReinforcementUnitStyle, SystemStyle; стили позволяют получать расширенную информацию об объекте, см. соответствующие им интерфейсы Renga;
- класс RebarUsage для работы с использованием арматуры (свойства и геометрия);

Удалено
- нод Project.IModel, так как у него был дуюлирующий нод (класс Model (Project project));
